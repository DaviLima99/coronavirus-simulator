## 💾 Getting Started
**Cloning the Repository**

```
$ git clonehttps://github.com/DaviLima99/coronavirus-simulator

```

**Installing dependencies and run**

```
$ cd coronavirus-simulator/

$ yarn
```

_or_

```
$ npm install

$ npm start
```
